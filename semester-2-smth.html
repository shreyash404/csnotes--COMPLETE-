Statistical Methods and Testing of Hypothesis


Statistical methods and testing of hypothesis are important subjects in computer science because they allow researchers and practitioners to analyze and interpret data, make predictions about future outcomes, and draw conclusions about the relationships between different variables.


Statistical methods are techniques for collecting, analyzing, and interpreting data. They involve the use of statistical models and algorithms to draw inferences from data, and they are often used to make predictions about future trends or to identify patterns and correlations in data sets.


Testing of hypothesis is a statistical procedure used to evaluate the likelihood that a hypothesis or theory is true. It involves formulating a hypothesis, collecting data, and then using statistical tests to determine whether the data support the hypothesis. If the data support the hypothesis, it is considered to be statistically significant, and the hypothesis is considered to be supported by the data.


Overall, statistical methods and testing of hypothesis are essential tools for researchers and practitioners in computer science because they allow them to analyze and interpret data, make predictions, and draw conclusions about the relationships between different variables.

A random variable is a variable in probability and statistics that represents the possible values of a random event. The term "random" refers to the fact that the values of the variable are not determined in advance and are dependent on the outcome of some chance process. In mathematical terms, a random variable is a function that maps outcomes of a probability space to real values.

In statistics, random variables are often used to model real-world phenomena, such as the height of a person, the length of time it takes to complete a task, or the value of a stock at a given time. The distribution of a random variable, which describes the likelihood of different values being observed, can be described using probability distributions, such as the normal or the Poisson distribution.

Random variables can be either discrete or continuous, depending on whether the possible values form a finite or infinite set. Discrete random variables can take on only a finite or countably infinite number of values, while continuous random variables can take on any value within a specified range.

A discrete random variable is a type of random variable that can only take on a finite or countably infinite number of values. These values are usually integers or values that can be expressed as integers, such as 0, 1, 2, etc.

Discrete random variables are often used to model situations where the outcomes are countable and can be assigned probabilities. Examples of discrete random variables include the number of heads in a series of coin flips, the number of calls received by a call center in an hour, and the number of defective items in a batch of goods.

The distribution of a discrete random variable is often described using a probability mass function (PMF), which assigns a probability to each possible value that the random variable can take on. The sum of all the probabilities in a PMF must equal 1, since one of the possible values must occur.

Discrete random variables can be used to calculate various measures of interest, such as the expected value (also known as the mean), variance, and standard deviation. These measures can help to describe the behavior of the random variable and make predictions about future outcomes.

A continuous random variable is a random variable that can take an uncountably infinite number of possible values within a given range, such as height, weight, or time. The values of a continuous random variable can be any value within the specified range, and the probabilities are defined in terms of continuous probability density functions, rather than discrete probability mass functions.

For example, the height of a person is a continuous random variable because it can take any value within a certain range, such as between 5 feet and 6 feet. The exact height of a person can be expressed as a decimal value, and there are infinitely many possible values within the range, such as 5.5 feet, 5.6 feet, 5.7 feet, and so on.

Continuous random variables are typically used to model real-world phenomena that exhibit smooth, continuous behavior, such as physical measurements, time, or space. The distributions of continuous random variables can be described using continuous probability density functions, such as the normal distribution or the exponential distribution.

Probability distribution is a function that describes the likelihood of obtaining the possible values that a random variable can take. It provides a complete description of the randomness of a random variable, including the range of values it can take, the average value it is expected to take, and the shape of its distribution.

There are two types of probability distributions: discrete probability distributions and continuous probability distributions. Discrete probability distributions describe the distribution of discrete random variables, such as the number of heads in a coin flip. Continuous probability distributions describe the distribution of continuous random variables, such as the height of a person.

The most commonly used probability distributions include the normal distribution, the binomial distribution, the Poisson distribution, and the exponential distribution. These distributions are useful for modeling different types of random events and for making predictions about the behavior of random variables in various applications.

CDF, or cumulative distribution function, of a discrete random variable gives the cumulative probability of getting values less than or equal to a certain value. In other words, it is a function that maps values of a random variable to probabilities, showing the cumulative sum of probabilities for all values less than or equal to a specific value. The CDF of a discrete random variable can be either calculated using a mathematical formula or obtained from the cumulative frequency distribution of the random variable.

Probability Density Function (PDF) is a function that describes the continuous distribution of a random variable. The PDF assigns a probability value to each interval of values of the random variable, showing how likely the random variable is to take a value within that interval. It is a mathematical representation of the continuous probability distribution of a random variable and provides a way to visualize the distribution of the variable. The area under the PDF curve between two points represents the probability that the random variable falls within that interval. PDFs are often used in statistics and probability theory to describe the distribution of continuous variables, such as height, weight, or time.

Expectation or mean of a probability distribution is a measure of central tendency that gives the expected value of the random variable. It is calculated as the weighted average of all possible values of the random variable, with their probabilities as weights.

Variance of a probability distribution is a measure of dispersion that gives the spread of the random variable around its mean. It is calculated as the average of the squared deviations of the random variable from its mean. The square root of the variance is called standard deviation. Variance and standard deviation help to quantify the uncertainty in the random variable.
