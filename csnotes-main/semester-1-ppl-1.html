Programing with python-1


Python in Computer Science.


Python is a high-level, general-purpose programming language that is widely used in the field of computer science. It is known for its simplicity, readability, and flexibility, making it a popular choice for beginners and experienced programmers alike.


In computer science, Python is often used for a variety of purposes, including:


Web development: Python has a number of libraries and frameworks that make it easy to develop web applications, such as Django and Flask.


Data analysis and machine learning: Python has a number of libraries and tools for data manipulation, visualization, and machine learning, such as Pandas, NumPy, and scikit-learn.


Scientific computing: Python has a number of libraries for scientific computing, such as NumPy and SciPy, which are commonly used in fields such as physics, chemistry, and biology.


Game development: Python has a number of libraries and frameworks for game development, such as Pygame and Pyglet.


Overall, Python is a versatile and widely used programming language in the field of computer science, and it is an essential tool for many professionals in this field. It is a great choice for students looking to learn programming or to build projects in computer science.

  • Readability : Python code is often said to be easy to read and understand, due to its use of indentation to delimit blocks of code, and its use of English keywords.
  • Dynamically-typed : In Python, variables do not have to be declared with a specific type, and their type can change at any point in the program. This allows for greater flexibility, but can also make it more difficult to catch type-related errors.
  • High-level : Python is a high-level programming language, which means that it abstracts away many of the details of the computer's hardware and operating system. This makes it easier to write and understand code, but also means that Python programs may not be as efficient as programs written in a lower-level language.
  • Object-oriented: Python supports object-oriented programming, which means that it allows programmers to define their own types (called "classes") and create objects based on those classes.
  • Extensive standard library : Python comes with a large standard library that includes modules for many common programming tasks, such as connecting to web servers, reading and writing files, and working with data.
  • Interpreter : Python is an interpreted language, which means that it does not need to be compiled before it is run. This makes it easy to try out small bits of code, but also means that Python programs may run more slowly than programs written in a compiled language.
  • Portable : Python programs can run on any computer that has an interpreter, making it an ideal language for writing cross-platform software.
  • Extensible : It is easy to write Python code that can be called from other programming languages, or to include code written in other languages in a Python program. This makes it easy to integrate Python with other tools and systems.
  • Free and open-source : Python is freely available and can be used and distributed without charge. Its source code is also available, which means that anyone can view or modify it.

    • IDLE (Integrated Development and Learning Environment) is an integrated development environment (IDE) for Python. It is included with the standard distribution of Python and is a convenient way to write and test Python code.

    IDLE includes a text editor that allows you to write Python code, as well as a shell window where you can run your code and see the results. It also includes debugging tools and other features that can be useful when writing and testing Python code.

  • Variables: A variable is a named location in memory where a programmer can store data. In Python, variables do not need to be declared with a specific type, and their type can change at any point in the program.
  • Operators: Operators are special symbols in Python that perform specific operations on one or more operands. For example, the addition operator `+` adds two operands together, and the multiplication operator `*` multiplies two operands.
  • Expressions: An expression is a combination of values, variables, and operators that represents a single result. For example, `2 + 3` is an expression that evaluates to the value 5.
  • Statements: A statement is a unit of code that performs a specific task. In Python, statements are typically terminated with a newline or a semicolon.
  • Indentation: Python uses indentation to delimit blocks of code, such as those within a function or loop. This helps to make the code more readable and easier to understand.
  • Comments: Comments are lines of code that are ignored by the Python interpreter and are used to add explanations or notes to the code. In Python, comments are indicated by a `#` symbol.
  • Functions: A function is a block of code that performs a specific task and can be called from other parts of the program. Functions can accept arguments (also called parameters) and can return a value when they are finished.
  • Modules: A module is a file containing Python code that can be imported into other Python programs. This allows programmers to reuse code and build larger programs by combining smaller modules.
  • Objects: In Python, everything is an object. Objects have attributes (data) and methods (functions that operate on the data) and can be created from classes (templates for objects).

    • In Python, a variable is a named location in memory where a programmer can store a value. This value can be of various types, such as a string, a number, or a list.

    Here are some rules for defining variables in Python:
    1. Variable names can only contain letters, numbers, and underscores. They cannot start with a number.

    2. Variable names are case-sensitive, so my_var and my_Var are different variables.

    3. Python reserves a set of keywords that cannot be used as variable names.

    4. Variable names should be descriptive and meaningful, but they can be as long as you like. It is generally a good idea to use descriptive names that are easy to understand, rather than short, abbreviated names.

    5. Variable names should not be the same as any Python built-in function or module names.

    6. It is recommended to use lowercase_with_underscores for naming variables in Python. This is known as "snake case" and is the most commonly used naming convention in Python.

  • Arithmetic operators: These operators perform basic arithmetic operations, such as addition, subtraction, multiplication, and division. For example: `+`, `-`, `*`, `/`, `%`, `**` (exponentiation), `//` (floor division).
  • Comparison operators: These operators compare two values and return a boolean value indicating whether the comparison is true or false. For example: `==` (equal to), `!=` (not equal to), `<` (less than), `>` (greater than), `<=` (less than or equal to), `>=` (greater than or equal to).
  • Assignment operators: These operators assign a value to a variable. For example: `=`, `+=`, `-=`, `*=`, `/=`, `%=`, `**=`, `//=`.
  • Logical operators: These operators perform boolean operations on two boolean values. For example: `and`, `or`, `not`.
  • Bitwise operators: These operators perform bit-level operations on integers. For example: `&` (bitwise AND), `|` (bitwise OR), `^` (bitwise XOR), `~` (bitwise NOT), `<<` (left shift), `>>` (right shift).
  • Membership operators: These operators test whether a value is a member of a sequence (such as a list or a string). For example: `in`, `not in`.
  • Identity operators: These operators test whether two values are the same object in memory. For example: `is`, `is not`.

    • Strings in Python are also used to store text data and are often used to represent words, sentences, or paragraphs. You can create a string by enclosing a piece of text in single or double quotes.

    In Python, strings are immutable, which means that once you create a string, you cannot change it. This is because strings are stored in memory as a series of characters, and changing any of those characters would require creating a new string and storing it in memory, while the original string still takes up space. This is different from mutable data types, such as lists or dictionaries, which can be modified in place.

    In Python, a comment is a piece of text in a program that is not executed by the interpreter.
    Comments are used to explain what the code does, or to leave notes for other people who might read the code.

    In Python, you can create a single-line comment by starting a line with the # symbol. Anything after the '#' on the same line will be considered a comment and ignored by the interpreter.

    You can also create a multi-line comment by using triple quotes, either ''' or """. Anything between the triple quotes will be considered a comment, and can span multiple lines.

    In Python, a list is an ordered collection of objects that can be of different types, including other lists. Lists are a versatile data type that allow you to store and manipulate data in a variety of ways.

    Here are some examples of common operations you can perform with lists in Python:
    Create a list:

    my_list = [1, 2, 3]
    my_list = ['a', 'b', 'c']

    In Python, a tuple is an immutable sequence data type. This means that once you create a tuple, you cannot change the values it contains: you cannot add, remove, or modify the values of any of the elements in the tuple.

    Tuples are similar to lists, but they are created using parentheses '()' instead of square brackets '[]'. You can create a tuple by enclosing a sequence of values in parentheses,
    like this:

    # Create a tuple with multiple elements
    t = (1, 'a', 3.14)

    # Create an empty tuple
    empty_tuple = ()

    # Create a tuple with a single element
    one_element_tuple = (42,)

    In Python, a dictionary is a collection of key-value pairs.
    It is a flexible data structure that allows you to store and retrieve data efficiently by using keys to access the values.

    Here are some examples of common operations you can perform with dictionaries in Python:

    Create a dictionary:
    my_dict = {'a': 1, 'b': 2, 'c': 3}
    my_dict = dict(a=1, b=2, c=3)
    my_dict = dict([('a', 1), ('b', 2), ('c', 3)])

    If-Elif-Else Statements in Python

    In Python, if-elif-else statements are used to execute different blocks of code depending on whether a condition is True or False.



    The basic structure of an if-elif-else statement is as follows:

          if condition1:
          # execute code block 1
      elif condition2:
          # execute code block 2
      else:
          # execute code block 3

    If condition1 is True, then the code block under the if statement is executed. If condition1 is False and condition2 is True, then the code block under the elif statement is executed. If both condition1 and condition2 are False, then the code block under the else statement is executed.

    It's important to note that you can have as many elif statements as you want in an if-elif-else block, and the else statement is optional.

    Looping in Python

    In Python, looping allows you to execute a block of code repeatedly for a given number of times or until a certain condition is met. There are two types of loops in Python: for loops and while loops.


    A for loop iterates over a sequence of objects, such as a list or a string. The basic structure of a for loop is as follows:

          for item in sequence:
          # execute code block

    A while loop continues to execute as long as a certain condition is True. The basic structure of a while loop is as follows:

          while condition:
          # execute code block

    It's important to note that you should use caution when using loops, as an infinite loop can cause your program to crash. To prevent this, make sure to include a way to break out of the loop once the desired condition has been met.
