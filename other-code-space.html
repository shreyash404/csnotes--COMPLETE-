Welcome to Code Space

This is a platform where you can share your code with the community and also view code written by others. To share your code, please follow the instructions below:

  1. Enter your full name in the "Your Name" field or you can enter subject name along with AkA.
  2. Enter a title for your program in the "Program Title" field. Please make sure the title consists of more than one word.
  3. Select the programming language you used for your program from the "Program Language" dropdown menu.
  4. Paste the code of your program in the "Program Code" field.
  5. Click the "Post" button to submit your program. You will be taken to the top of the page after your program is submitted.

Click "Post Code" button to create your new post!
